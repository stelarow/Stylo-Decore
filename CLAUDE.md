# Stylo Decore - Instruções para Claude Code

## Skills Automáticos

Use os skills abaixo **proativamente**, sem esperar o usuário pedir explicitamente.

### `/frontend-design`
Invoque sempre que a tarefa envolver:
- Criar ou modificar componentes, páginas ou layouts
- Melhorar estética, tipografia, cores ou animações
- Construir qualquer UI (cards, heroes, grids, formulários, etc.)

### `/programmatic-seo`
Invoque sempre que a tarefa envolver:
- Criar páginas em escala (ex: por categoria, subcategoria, cidade)
- Estrutura de URLs, sitemaps, meta tags, schema markup
- Estratégia de conteúdo para ranqueamento orgânico

### `/skill-creator`
Invoque sempre que o usuário quiser criar ou atualizar um skill personalizado para o projeto.

## Contexto do Projeto

Showroom digital de decoração (cortinas, persianas, papel de parede, tapetes) em Florianópolis/SC.
Funil: navegação inspiracional → WhatsApp consultivo. Sem e-commerce.

Stack: Next.js, React 19, TypeScript, Tailwind CSS v4.
Design: Manrope + Playfair Display, dourado #f1bf27, fundo #F1EADA, mahogany #584738.
